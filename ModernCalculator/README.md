# 🧮 Smart Scientific Calculator

A Python GUI calculator built using CustomTkinter that supports both basic and scientific calculations with a modern interface.

## 🚀 Features
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, log, sqrt, etc.)
- Calculation history with timestamps
- Dark/Light theme toggle
- Copy result feature
- Backspace and Clear buttons

## 🛠️ Tech Stack
- Python
- CustomTkinter
- Math module
- Datetime module

## ▶️ Run Project
```bash
pip install customtkinter
python calculator.py
